## Why Medix?

Medix wraps FFmpeg in an interactive CLI that makes batch conversion actually
pleasant. Point it at a file or a folder, pick an output format, and watch the
progress bars fly.

<CardGrid stagger>
  <Card title="Interactive-first" icon="rocket">
    Questionary-powered prompts guide you through source filtering, format
    selection, and advanced encoding tuning.
  </Card>
  <Card title="Zero config" icon="setting">
    No dotfiles, no YAML. All choices happen at runtime. Sensible defaults
    per output format.
  </Card>
  <Card title="Auto-install FFmpeg" icon="puzzle">
    Detects your OS and offers to install FFmpeg via Homebrew, APT, DNF,
    Pacman, winget, Chocolatey, or Scoop.
  </Card>
  <Card title="Dry run first" icon="approve-check">
    Preview the exact ffmpeg commands before anything touches disk with
    `--dry-run`.
  </Card>
  <Card title="Progress you trust" icon="bars">
    Per-file and overall progress bars with ETA, powered by Rich.
  </Card>
  <Card title="Safe by default" icon="seti:lock">
    Output filename collisions get auto-numbered. Nothing is overwritten
    without your say-so.
  </Card>
</CardGrid>
